## Architecture

**Stack:** React 18 + TypeScript + Vite, Tailwind CSS, React Router v6, Axios, lucide-react.

### Data flow

`HueContext` (`src/context/HueContext.tsx`) is the single source of truth. It polls the Hue bridge every 10 seconds and exposes lights, groups, and scenes. All mutations (toggle, brightness, scene activation) apply an **optimistic update** to local state before awaiting the API call, then re-fetch on group-level changes. Bridge config (IP + username) is persisted to `localStorage` under the key `hue_bridge_config`.

### Hue API layer (`src/api/hue.ts`)

- All bridge calls go through `createClient(config)` which builds an Axios instance at `http://{ip}/api/{username}`.
- UI brightness is 0–100%; the API uses 1–254. `briToPercent` / `percentToBri` convert between them.
- `hueStateToColor(light)` converts the light's current state to a CSS color, handling all three Hue color modes: `hs` (hue/sat → HSL), `xy` (CIE XYZ → sRGB via matrix), and `ct` (mirek → interpolated warm/cool RGB). This color is used to tint cards and sliders.

### Routing

| Route        | Page                                                                                       |
| ------------ | ------------------------------------------------------------------------------------------ |
| `/`          | Dashboard — stats, room cards, light carousel, quick scenes                                |
| `/lights`    | All lights with search + room + active filters                                             |
| `/scenes`    | Scenes grouped by room                                                                     |
| `/rooms`     | All rooms                                                                                  |
| `/rooms/:id` | Room detail — per-room stats, individual light cards, group brightness slider, room scenes |
| `/routines`  | Placeholder (Hue schedules API not yet implemented)                                        |
| `/settings`  | Bridge IP/username config, reset                                                           |

`RoomCard` navigates to `/rooms/:id` by default (no `onClick` prop required).
